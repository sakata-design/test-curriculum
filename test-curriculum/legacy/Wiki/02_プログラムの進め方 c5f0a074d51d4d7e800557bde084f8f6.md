# 02_プログラムの進め方

# テストプログラムの**構成**

- テストケース(IT1、IT2)

　└ [結合試験](%E7%B5%90%E5%90%88%E8%A9%A6%E9%A8%93%209aac211cd393454abc16ffae21c8c77a.md)ページ配下にあなたの名前ページがあります。

　　→ ページにあるGoogleドライブの中にテスト項目が書かれているIT1、IT2という
　　　スプレッドシートがあるのでそちらを見ながらテストの実施を行っていただきます。

- エビデンス

　└ テストを実施した項目に対して期待値に当たる画面のスクリーンショットを取得し、
　　 Googleドライブにある「エビデンス」フォルダにエビデンスを格納してください。

　└ スクリーンショットの名前はテストケースの項目名と同じにしてください
　　 (例：IT1_001)エビデンスが複数枚になる場合は(例 : IT1_001_1、IT1_001_2)とする。
